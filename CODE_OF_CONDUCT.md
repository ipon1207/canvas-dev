# 行動規範 (Code of Conduct)

## 私たちの誓い

私たちは、コミュニティへの参加を、年齢、体格、障害、民族、性自認と性表現、経験レベル、国籍、個人の外見、人種、宗教、性的指向に関わらず、全員にとってハラスメントのない体験にすることを誓います。

私たちは、オープンで、歓迎され、多様で、包摂的で、健全なコミュニティを育むような方法で交流し、行動することを誓います。

## 私たちの基準

コミュニティにプラスとなる振る舞いの例：

* 共感と優しさを示すこと
* 異なる意見、視点、経験を尊重すること
*建設的なフィードバックを与え、優雅に受け入れること
* 謝罪が必要な過ちを犯したときは責任を取り、そこから学ぶこと
* 個人のことだけでなく、コミュニティ全体にとって最善のことを考えること

許容されない振る舞いの例：

* 性的言語や画像の使用、およびあらゆる種類の性的注目や求愛
* トローリング、侮辱的・軽蔑的なコメント、個人的または政治的な攻撃
* 公的または私的な嫌がらせ（ハラスメント）
* 許可なく他人の身体的特徴や個人情報（住所、メールアドレスなど）を公開すること
* その他、職業的な場において不適切と見なされる行為

## 執行の責任

プロジェクトのメンテナは、許容される行動の基準を明確にし、不適切、脅迫的、攻撃的、または有害な行動に対して、適切かつ公正な是正措置を講じる責任があります。

メンテナは、この行動規範に従わないコメント、コミット、コード、Wikiの編集、課題、およびその他の貢献を削除、編集、または拒否する権利と責任を持ちます。また、不適切と見なされる行動を行った貢献者を、一時的または恒久的に追放する権利を持ちます。

## 適用範囲

この行動規範は、コミュニティのスペース内、およびコミュニティを代表する個人が公的な場にいる際にも適用されます。

## 報告

虐待、ハラスメント、またはその他の許容できない行動があった場合、プロジェクトのメンテナに連絡して報告してください。すべての苦情は迅速かつ公正に調査されます。

メンテナは、報告者のプライバシーと安全を尊重する義務があります。

## 帰属

この行動規範は [Contributor Covenant](https://www.contributor-covenant.org), version 2.1 に基づいています。
